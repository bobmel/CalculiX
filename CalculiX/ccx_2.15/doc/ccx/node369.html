Network MPC.

By using the subroutines networkmpc_lhs.f and networkmpc_rhs.f in combination with the keyword *NETWORK MPC the user can define an arbitrary linear or nonlinear equation between the degrees of freedom in a network, i.e. the total temperature, total pressure and mass flow. For details the user is referred to the source code and the *NETWORK MPC keyword.


Example file: networkmpc.
